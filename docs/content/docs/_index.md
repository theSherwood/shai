Welcome to the Shai documentation!

Shai is a sandboxing tool for running CLI-based AI agents inside containers with fine-grained control over filesystem access, network connectivity, and system resources.

## Getting Started

New to Shai? Start here:

- [Quick Start](/docs/quick-start) - Install Shai and run your first sandbox
- [Core Concepts](/docs/concepts) - Understand how Shai works
- [Examples](/docs/examples) - See real-world usage patterns
